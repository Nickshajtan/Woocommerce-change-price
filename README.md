# HCC Woocommerce simple plugin
version 1.0.0 

### Description

My special implement [backend test task](test.pdf) in backend directory and [frontend test task](Frontend.pdf) in frontend directory for Yozma tech company.

## First part: backend
-----------------------------------------------------
This plugin create "HCC WC addons" theme subpage with filds for change WC product price.
For changing used Ajax POST.
## Second part: backend
-----------------------------------------------------
File "functions.php" contain some functions for creating and sorting new columns in Wordpress Users page.
## Third part: frontend
-----------------------------------------------------
Dist directory is dir for development files, public dir for output result files.
Gulp folder is directory for gulp 4 common js modules.
